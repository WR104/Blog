fix_pyopencl_import_error

Zhenyu Jia
2024-02-03

I’m on Windows 10 running python 3.12.1 and encounter DLL load failed error >ImportError: DLL load failed while importing _cl: 找不到指定的程序。

Download the opencl.dll file from google.

Go to the location where the pyopencl library is installed, specifically where the tools.py is, and paste the DLL file there.
